# 总结：

用let声明的变量特点：

  • 只在块级作用域起作用

  • 不会出现变量提升现象

  • 同一个代码块内，不可以重复声明相同的变量

# const

const和let一样也是用来声明变量的，但是const是专门用来声明一个常量的

常量的特点：

1、不可修改（引用类型的时候不能修改引用空间）

  • 如果是基本数据类型
const a = 10;
a = 20;
1
2

运行结果如下：

1

会出现一个错误提示，原因是常量不能被修改

  • 如果常量是一个对象
const a = {name:'lisi'};
a.age = 9;
console.log(a)
